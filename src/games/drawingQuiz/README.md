# 그림 퀴즈 게임

## 게임 개요

그림 퀴즈는 모든 플레이어가 돌아가며 주어진 단어를 그리고, 다른 플레이어들이 채팅을 통해 정답을 맞히는 게임입니다.

### 기본 정보
- **최소 플레이어**: 2명
- **최대 플레이어**: 12명
- **한 라운드 시간**: 90초
- **총 라운드**: 플레이어 수 (모두가 한번씩 그림)

## 게임 규칙

### 라운드 진행
1. 각 라운드마다 한 명의 플레이어(아티스트)가 주어진 단어를 그립니다.
2. 다른 플레이어들은 그림을 보고 채팅으로 정답을 제출합니다.
3. 정답을 맞춘 순서에 따라 점수가 주어집니다.
4. 라운드 종료 후 다음 플레이어가 아티스트가 되어 반복합니다.

### 점수 산정

#### 정답자 점수
정답을 맞춘 순서에 따라 다음과 같이 점수를 획득합니다:

| 순위 | 점수 |
|------|------|
| 1등  | 100점 |
| 2등  | 80점  |
| 3등  | 60점  |
| 4등  | 50점  |
| 5등  | 40점  |
| 6등  | 30점  |
| 7등  | 20점  |
| 8등  | 10점  |
| 9등 이상 | 5점 |

#### 아티스트 점수
아티스트(그린 사람)는 다른 플레이어들이 정답을 맞출 때마다 점수를 획득합니다:
- **정답자 1명당 30점**
- **최대 100점** (정답이 4명 이상일 때)

### 채팅 규칙
- 정답을 맞춘 플레이어의 메시지는 게임 진행 채팅에 정상 표시됩니다.
- 오답을 제출한 플레이어의 메시지는 "(마스킹됨)"으로 표시되어, 다른 플레이어가 정답 힌트를 얻지 못하도록 합니다.

## 게임 상태 전환

```
LOBBY → IN_GAME (라운드 1) → IN_GAME (라운드 2) → ... → IN_GAME (라운드 N) → RESULT
```

## 결과 화면

게임이 종료되면 플레이어들의 최종 점수를 기준으로 순위를 매겨 표시합니다.

## 향후 개선사항

- [ ] 단어 카테고리 추가 (쉬움/보통/어려움)
- [ ] 힌트 시스템 (일정 시간 경과 후 자동 제공)
- [ ] 단어 유사도 검사 (완벽하지 않은 정답도 수용)
- [ ] 라운드별 상세 통계
